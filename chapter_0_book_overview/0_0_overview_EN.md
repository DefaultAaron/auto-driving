# Chapter 0 — Book Overview

> [!info] Status & canonical state
> Frontmatter `status` is canonical. [[00_table_of_contents|TOC]] badges are a manual display copy.

> [!abstract] What this chapter covers
> Why this book exists, what it covers, who it is for, how it is structured (pedagogically — not as a project diary), and how to use the bilingual EN/ZH layout, the reading list, and the running mingtai traffic-light example.

## Learning objectives
- Understand the book's scope: a comprehensive guide to mastering autonomous driving end-to-end.
- Know how to navigate the chapters, table of contents, and reading list.
- Understand the role of the mingtai traffic-light project as a worked example, not a structural anchor.
- Set expectations for what is covered (foundations + novel work, perception → planning → safety) and what is not.

## Prerequisites
- None. Some comfort with Python, Linux, and undergraduate linear algebra / probability is assumed for later chapters.
