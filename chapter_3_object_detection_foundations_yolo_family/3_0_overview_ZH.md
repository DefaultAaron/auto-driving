# 第 3 章 — 目标检测基础与 YOLO 思想脉络

> [!info] 状态与权威来源
> Frontmatter 中的 `status` 字段是权威来源。[[00_table_of_contents|目录]] 中的状态标记为手动同步的展示副本。

> [!abstract] 本章内容
> 目标检测的**理论**机制:框、标签、IoU、损失、数据、指标、失败分析 —— 以及通过 YOLOv1 网格直觉引入的**单阶段检测思想**。家族沿革做简要指引以便读者知道在哪深入,但**完整作为部署系统的 YOLO 家族**(含用户的 "YOLO26" 项目标签与 mingtai 红绿灯走读)放在 [[4_0_overview_ZH|第 4 章 — 面向自动驾驶的相机感知系统]]。

## 学习目标
- 准确描述目标检测问题(框参数化、标签格式、IoU、置信度)。
- 推理分类 + 定位损失、focal loss、IoU 类损失。
- 构建、增强并调试检测数据集。
- 用 mAP / PR 曲线评估检测器,读懂失败模式。
- 通过 YOLOv1 的网格直觉理解单阶段检测思想;知道家族沿革从这里走向何方(深度留给第 4 章)。
