# 第 1 章 — 基础知识

> [!info] 状态与权威来源
> Frontmatter 中的 `status` 字段是权威来源。[[00_table_of_contents|目录]] 中的状态标记为手动同步的展示副本。

> [!abstract] 本章内容
> 后续所有章节都假设读者已掌握的共同基础:坐标系与变换、相机 / 激光雷达标定、传感器时间同步、ROS2 基础、评估方法、数据集与数据版本管理、车辆动力学、本书所对标的部署目标(PyTorch → ONNX / TensorRT → C++ ROS2 节点 → 车端边缘 GPU),以及安全相关概念(ODD、危害分析)的初步介绍 —— 第 10 章会展开。

## 学习目标
- 能够推理坐标系、变换以及 TF2。
- 完成相机与激光雷达标定,并能解释由此得到的投影关系。
- 理解传感器时间同步、ROS2 消息流与 rosbag 录制 / 回放。
- 在 AD 问题中使用合理的评估方法与数据卫生原则。
- 理解本书对标的部署目标。
- 对 ODD 与基础危害分析有足够认识,以便阅读后续每章的"安全"小节。

## 先修要求
- [[0_0_概览_ZH|第 0 章 — 本书概览]]
- 熟悉 Python 与 Linux;本科线性代数与概率。
