# 第 5 章 — 经典激光雷达检测

> [!info] 状态与权威来源
> Frontmatter 中的 `status` 字段是权威来源。[[00_table_of_contents|目录]] 中的状态标记为手动同步的展示副本。

> [!abstract] 本章内容
> AD 团队多年来在用、且至今仍在用的经典(非 DL)激光雷达检测工具箱:点云预处理、地面 / 非地面分割、聚类、多目标跟踪、配准(定位章节前向指针指向这里)、占据 / 可行驶空间推理、ROS2 集成,以及让一切真正能跑起来的部署 / 运行时约束。本章是 [[6_0_overview_ZH|第 6 章 — 深度学习激光雷达检测]] 将要替换或混合的基线。

## 学习目标
- 对原始点云进行预处理(下采样 / 体素滤波 / 离群点剔除)。
- 用 RANSAC 类方法做地面分割,并理解其失败模式。
- 用 Euclidean / DBSCAN 对非地面点聚类得到目标候选。
- 用 Kalman / IMM / JPDA 实现多目标跟踪。
- 用 ICP / NDT / GICP 配准多帧或多激光雷达扫描 —— [[2_0_overview_ZH|第 2 章 §2.3]] 激光雷达定位的算法深度就在这里。
- 构建占据栅格、估计可行驶空间。
- 将整条管线封装为 ROS2 节点 / topic。
- 推理部署约束(CPU/GPU 预算、延迟、ROS2 时序、现场鲁棒性)。
